# Cybersecurity-Porfolio & IT Automation Portfolio
This repository contains practical scripts and labs developed while training for entry-level IT and Security roles.

## 🛠️Project 1: Log Crusher (Bash)
**Description:** A custom Bash script designed to automate security monitoring by scanning Linux authentication logs.
**Problem Solved:** Manually reviewing thousands of log lines to find security threats is inefficient.
**How it Works:** The script uses `grep` and `awk` to identify failed login attempts and outputs a unique list of suspicious IP addresses.
**Key Skills:** Linux CLI, Log Analysis, Scripting.

## 🛠️ Project 2: Network Asset Scanner (Python)
**Description:** A Python-based tool for discovering active devices on a local network.
**Problem Solved:** Identifying "shadow IT" or unauthorized devices connected to a corporate network.
**How it Works:** It uses the `subprocess` library to ping targets and reports their "Up/Down" status in real-time.
**Key Skills:** Python Automation, Networking Fundamentals (ICMP/Ping).
